# BPM Player

**Fast, simple audio player with BPM. Built by a DJ.**

Load a track and press play. Everything else stays out of the way until you need it.

---

## Features

**Fast and responsive**  
5.9 MB on disk. Tracks play instantly. BPM is only analysed when needed — using a short 60-second segment instead of the full file. Already tagged? The app reads the ID3 BPM field and skips analysis entirely.

**BPM & speed control**  
Tempo detection runs only if enabled. Adjust playback speed from 50% to 200% using high-quality [Rubber Band](https://breakfastquay.com/rubberband/) time-stretching. Key Lock keeps pitch stable at any speed.

**Playlist**  
Build a session playlist by dragging files or using the open panel. Import from M3U. Export to Rekordbox XML and Serato .crate on the fly. Tracks are saved between launches.

**Converter**  
Convert any track to FLAC (24-bit / 48 or 96 kHz), AIFF (24-bit / 48 or 96 kHz), or MP3 320 kbps. Progress updates in real time.

**Tag editor**  
Edit title, artist, and album artwork directly inside the app. Changes are written back to the file in the original format. Artwork is resized to 800×800 JPG for compatibility with DJ equipment.

**Formats**  
MP3, FLAC, AIFF, WAV, M4A, AAC, Apple Lossless (ALAC), OGG, and more.

**No subscription. No ads. No tracking.**  
Free. No network requests. No data collection of any kind.
